# Tasca 1: Joc 3D 
 
Projecte per a la modificació d'un entorn 2D a un plànol 3D, pertanyent al Mòdul 0490: Programació de serveis i processos. 
 
## Estructura del projecte 
- **EventJoc**: Gestiona els hils de cada element i s'ha implementat la profunditat (posició Z) per simulacions 3D aleatòries. 
- **FaseJoc**: Classe principal que inicialitza els elements del joc i els executa de manera simultània.
